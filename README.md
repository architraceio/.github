# architrace

A decision-mesh platform that reconciles ADRs, Confluence pages,
READMEs, and AI agent rule files into one source of truth across an
organization. Currently in early access — [architrace.io](https://architrace.io).

## Open source

- **[chronicle](https://github.com/architraceio/chronicle)** — capture
  architectural decisions as ADRs while you code with Claude Code or
  OpenCode. Sponsored by architrace; works fully offline; no account
  required.
